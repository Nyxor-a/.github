## 🛠️ What We've Built

### 🧠 Nexa — Multilingual Conversational Intelligence Platform
An AI-powered conversational platform built for Indian healthcare and finance workflows. Supports 11 Indian languages via Sarvam AI's STT/TTS stack, with Groq-powered LLM extraction and a Next.js 14 frontend. The routing logic to correctly separate display language from extraction language was the messiest bug we've fixed together.

**Stack:** Next.js 14 · FastAPI · Groq · Sarvam AI · Supabase

---

### 🏥 ArogyaDeep — Multilingual AI Health Assistant
Built for **Hack2Future (GIT 2025)**. A web app that takes symptom input, runs it through Gemini/OpenAI, and returns probable conditions, prevention tips, and nearby doctor listings — all with multilingual support. Deployed and live.

**Stack:** HTML/CSS/JS · Flask · Gemini API · OpenAI API  
🔗 [Live Demo](https://arogyadeep-backend.onrender.com/)

---

### 🏙️ Vspace — Collaborative City Planning Web App
An interactive city planning tool built as a web app. Lets users collaboratively plan and visualize urban spaces. Deployed on Vercel.

**Stack:** JavaScript · CSS · Firebase Firestore  
🔗 [Live Demo](https://vspacee.vercel.app/) · [Repository](https://github.com/SushilChandaragi/Vspace)

---

### 🔍 TruthLens — Autonomous Deep Research Agent
A research agent that doesn't just answer questions — it investigates them. Built with a focus on epistemic framing over confidence scores. Uses Firecrawl for web scraping and Groq's llama-3.3-70b-versatile for synthesis.

**Stack:** React · Vite · Node.js · Express · Groq · Firecrawl
